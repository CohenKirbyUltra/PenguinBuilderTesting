# Variables+
## set [type] var to
sets a variable to a value
### Arguments
* type: menu of the type of var
    * local
    * global
* name: name of the var
* value: the value of the var
___
## get [type] var
gets a variable
returns the variable
### Arguments
* type: menu of the type of var
    * local
    * global
* name: name of the var
___
